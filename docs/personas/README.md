# Persona cards

Each file is **`{id}.json`** where `id` is the stable slug used with:

```bash
python -m carousel_agents run --input doc.txt --out run.json --audience-preset long_road_diagnosis
```

Override directory with env **`PERSONAS_DIR`** (default: `docs/personas`).

See **`docs/audience-and-personas-plan.md`** and **`_template.json`** for fields.
